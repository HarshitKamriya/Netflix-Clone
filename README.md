# Netflix Clone

This is a simple front-end Netflix Clone created using pure HTML, CSS, and a little JavaScript. The project replicates the basic look and feel of the Netflix homepage and includes a static login page with button interactions.

## Project Overview
This project focuses on building a clean and responsive user interface similar to Netflix. It includes:
- Responsive homepage layout
- Login page UI
- Buttons with JavaScript interactions
- Simple, clean, and modern design

## Technologies Used
- HTML5 for structure
- CSS3 for styling and layout
- JavaScript for basic button interactions and login page functionality

## Features
1. Hero Section  
   - Background image  
   - Headings similar to Netflix  
   - Call-to-action buttons  

2. Login Page  
   - Static login form  
   - Email and password input fields  

3. Responsive Design  
   - Works on desktop, tablet, and mobile  

4. JavaScript Interactions  
   - Button click effects  
   - Basic validation or simple scripts (if used)
